## Workflow 1: Value Equation Scorer

Score = (Dream × Likelihood) / (Time × Effort). Perfect = 1,000,000.
Tiers: 1M+=Grand Slam | 500k-999k=Elite | 100k-499k=Strong | 20k-99k=Decent | 5k-19k=Weak | <5k=Broken
Denominator (Time/Effort) has MULTIPLIED impact. Fix these first.
4 fastest levers: Unconditional guarantee → DFY element → 7-day quick win → Niche down ICP

Output: Score report with breakdown → denominator alert → weaknesses priority order → improvement scenarios → bottom line action
