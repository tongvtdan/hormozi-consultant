# Delivery Cube — Reference
**Source: $100M Offers Course Checklist V**

The Delivery Cube determines HOW your offer is delivered across 6 dimensions.
The right cube selection directly multiplies your Value Equation score.

---

## The 6 Dimensions

### Dimension 1 — Group Ratio
| Option | Value Impact | Best For |
|---|---|---|
| 1-on-1 | Highest perceived value | High-ticket, complex problems |
| Small group (2–10) | High value, more scalable | Coaching cohorts, masterminds |
| One-to-many (11+) | Lower per-person value | Courses, memberships, events |

### Dimension 2 — Effort Level (most important dimension)
| Option | Value Equation Impact | Effort Score |
|---|---|---|
| DIY (Do It Yourself) | Baseline | 0.6–0.9 |
| DWY (Done With You) | +2–4x | 0.3–0.5 |
| **DFY (Done For You)** | **Maximum — collapses effort** | **0.05–0.2** |

**DFY is the single highest-leverage Delivery Cube choice.** It drops the Effort denominator to near-minimum, which multiplies the overall score 3–10x.

### Dimension 3 — Support Type
| Option | Perceived Value |
|---|---|
| Dedicated expert (named person) | Highest |
| Live Zoom/phone on demand | Very high |
| Chat/SMS | High |
| Email | Medium |
| Community/forum | Lower |
| No support | Lowest |

### Dimension 4 — Consumption Type
| Format | Effort Score | Best For |
|---|---|---|
| In-person live | Lowest effort perceived | High-ticket transformations |
| Virtual live (Zoom) | Low effort | Coaching, consulting |
| Recorded video | Medium | Courses, training |
| Written (docs/PDFs) | Medium-high | References, playbooks |
| Audio | Medium | On-the-go consumption |

### Dimension 5 — Speed & Convenience
| SLA | Perceived Value |
|---|---|
| 24/7 access/response | Maximum |
| Same-day | Very high |
| 9-5 business hours | Standard |
| Scheduled only | Lower |

### Dimension 6 — The 10x / 1/10th Test
Use this as a design pressure test:
- **At 10x the price:** what additional elements would you add? (Add some of these now)
- **At 1/10th the price with MORE value:** what would you remove? (Remove these to simplify)

The ideal cube sits between these two extremes — premium enough to justify the price, simple enough to over-deliver.

---

## Cube Selection by Offer Type

| Offer Type | Recommended Cube |
|---|---|
| Agency / Done-For-You service | DFY + 1-on-1 + Dedicated support + 24/7 |
| High-ticket coaching | DWY + Small group + Zoom + Weekly live |
| Course/program | DIY or DWY + One-to-many + Recorded + Community |
| SaaS product | DIY + One-to-many + Chat + 24/7 async |
| Consulting retainer | DWY + 1-on-1 + Zoom + Business hours |

## Value Stack Pricing Guide by Modality

When building your value table (Layer 5 — Trim & Stack), reference market rates:

| Modality | Standalone Market Value |
|---|---|
| 1-on-1 coaching call (60 min) | $300–$1,500 |
| Group coaching call (12 sessions) | $3,000–$8,000 |
| Custom strategy document | $2,500–$8,000 |
| DFY implementation sprint | $5,000–$25,000 |
| Swipe file / template library | $500–$2,000 |
| Checklist or tool | $200–$1,000 |
| Community access (annual) | $500–$3,000 |
| Audit or assessment | $1,500–$5,000 |

Always sum standalone values for the Value Anchor before revealing the actual price.
The gap should be minimum 5:1. Grand Slam Offers often achieve 10:1 or higher.
