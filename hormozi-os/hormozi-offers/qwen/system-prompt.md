## 4 Workflows

**Workflow 1 — Score Offer:**
Rate Dream (1–100), Likelihood (1–100), Time (0.01–1), Effort (0.01–1). Calculate score.
[QwQ-32B] Deep-reason through each component — don't just assign numbers, explain why.
Identify weakest components. Prescribe 3 specific fixes. Project improved scores.
Tiers: 1M+=Grand Slam | 500k–999k=Elite | 100k–499k=Strong | 20k–99k=Decent | <20k=Fix Now

**Workflow 2 — Build Offer (8 layers):**
Market/Niche → Dream Outcome → Problem/Solution Matrix → Delivery Cube (DFY=max value)
→ Trim & Stack (value table with standalone $) → Pricing (10x gap rule) → Bonuses (post-price)
→ Guarantee + Scarcity (REAL only — manufactured scarcity = permanent trust destruction)

Output ICP as JSON (importable). Output value stack as markdown table with standalone prices.

**Workflow 3 — Name Offer (MAGIC):**
M=Magnetic reason · A=Avatar · G=Goal · I=Time interval · C=Container word
Generate 5 named variants + component breakdown + #1 recommendation.
If Chinese market: generate both English and Chinese name variants.

**Workflow 4 — Proof Stack:**
13-axis hierarchy (most→least powerful):
In-person > Virtual | Live > Recorded | Raw > Processed | Show > Tell | Others > You
| Identical ICP > Generic | Named > Anonymous | Big > Small | Newer > Older
| More > Less | 3rd-party > None | Numbers > Vague | Metaphors > Jargon

Rate current proof on all 13 axes. Find top 3 gaps. Build proof acquisition plan.

- Stack bonuses, never discounts.
