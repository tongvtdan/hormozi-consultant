# Alex Skills Plugin

**Alex Hormozi-style business scaling prompts — engineered to take any business from $10M to $100M.**

A high-leverage Cowork plugin containing 6 first-principles frameworks derived from Alex Hormozi's scaling philosophy. Each skill is a structured execution system, not generic advice.

---

## Overview

> "Most founders chase ideas. $100M founders study markets, solve expensive problems, build irresistible offers, dominate distribution, engineer virality, and scale with systems."

This plugin operationalizes all six pillars into actionable, repeatable frameworks.

---

## Skills

| Skill                    | Command                   | What It Does                                                              |
|--------------------------|--------------------------|---------------------------------------------------------------------------|
| Market Breakdown         | `/market-breakdown`      | TAM/SAM/SOM, top trends, underserved gaps, where money flows              |
| Problem Prioritization   | `/problem-prioritization`| Top 10 problems ranked by urgency × willingness to pay                    |
| Offer Creation           | `/offer-creation`        | ICP, value prop, 3-tier pricing, guarantee, competitive differentiation   |
| Distribution Domination  | `/distribution-domination`| 5 acquisition channels, content formats, daily execution, organic/paid mix|
| Viral Content Engine     | `/viral-content`         | 20 hooks, 10 formats, emotional trigger map, shareability mechanics       |
| Scale System             | `/scale-system`          | Automate, delegate, build systems, hiring roadmap, bottleneck anticipation|

---

## Usage

### Skills (trigger via natural language)
Each skill activates automatically when you describe what you need:

- "Analyze the market for online fitness coaching" → **Market Breakdown**
- "What are the most expensive problems in HR software?" → **Problem Prioritization**
- "Create an offer for B2B bookkeeping services" → **Offer Creation**
- "How do I reach 1M people with my agency?" → **Distribution Domination**
- "Build a content strategy for my e-commerce brand" → **Viral Content Engine**
- "Scale my consulting firm to $500K/month" → **Scale System**

### Commands (explicit invocation)
```
/alex-skills:market-breakdown [niche]
/alex-skills:problem-prioritization [industry]
/alex-skills:offer-creation [problem]
/alex-skills:distribution-domination [idea]
/alex-skills:viral-content [niche]
/alex-skills:scale-system [business description + revenue stage]
```

## Philosophy

Each skill encodes one of Hormozi's six core principles:

1. **Market Breakdown** — "Most founders chase ideas. $100M founders study markets."
2. **Problem Prioritization** — "Amateurs solve random problems. Experts solve expensive ones."
3. **Offer Creation** — "Good products sell. Great offers print money."
4. **Distribution Domination** — "No distribution = no business."
5. **Viral Content Engine** — "Content is the new distribution."
6. **Scale System** — "Most people hustle. Founders build systems."
