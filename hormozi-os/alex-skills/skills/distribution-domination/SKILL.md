# The Distribution Domination Plan

**Philosophy:** "No distribution = no business."

## Trigger Conditions

Activate when the user has a product, idea, or offer and needs a concrete plan to get it in front of large audiences fast — with a focus on leverage, not brute effort.

## Execution Protocol

1. Confirm the business idea or offer if not provided.
2. Identify the 5 highest-leverage acquisition channels for that specific idea.
3. Define content formats for each channel.
4. Build a daily execution cadence.
5. Recommend the organic-to-paid mix by stage.
6. Output as a step-by-step system — not advice.

## The Distribution Architecture

### Step 1: Channel Selection (Top 5)

Evaluate channels across 3 criteria:
- **Reach**: How many of the ICP can this channel access?
- **Speed**: How fast can this channel generate traction?
- **Leverage**: Can one piece of content/effort compound over time?

Common high-leverage channels by business type:
- **B2B SaaS**: LinkedIn organic, cold email, content SEO, podcast guesting, partner channels
- **Consumer / DTC**: Short-form video (TikTok/Reels), influencer seeding, paid social, community-led
- **Professional services**: LinkedIn, referral networks, YouTube, speaking/events
- **Marketplace**: SEO, paid acquisition, supply-side outreach, community infiltration

### Step 2: Content Format by Channel

For each of the 5 selected channels, define:
- **Format**: Short-form video / long-form article / email / DM / ad creative / webinar
- **Frequency**: Daily / 3x/week / weekly
- **Hook style**: Problem-aware / solution-aware / result-led
- **Call to action**: DM / link / book a call / free resource

### Step 3: Daily Execution Plan

Build a 7-day repeatable distribution cadence:

```
Monday:    [Channel A action] + [Channel B action]
Tuesday:   [Channel C action] + [Repurpose Monday's content]
Wednesday: [Outreach batch] + [Channel D action]
Thursday:  [Channel B new content] + [Community engagement]
Friday:    [Channel A action] + [Analytics review]
Weekend:   [Batch create content for next week]
```

### Step 4: Organic + Paid Mix

By stage:
| Stage           | Organic % | Paid % | Rationale                               |
|-----------------|-----------|--------|-----------------------------------------|
| Pre-revenue     | 90%       | 10%    | Validate message before spending        |
| $0–$10K/mo      | 70%       | 30%    | Amplify what's working organically      |
| $10K–$100K/mo   | 50%       | 50%    | Scale winners with paid                 |
| $100K+/mo       | 30%       | 70%    | Paid dominates; organic supports brand  |

### Step 5: Leverage Identification

For the user's specific idea, identify the ONE mechanism that creates compounding reach:
- **Referral loop**: Every customer brings X customers
- **Content flywheel**: Each piece builds audience that grows the next
- **Partnership amplifier**: Each partner opens their whole audience
- **Platform algorithm**: Platform rewards consistency with organic reach


## Quality Standard

The plan must be executable tomorrow morning. No vague suggestions like "post more content." Every action must have a channel, format, frequency, and measurable output.
