## The Lead Magnet Purpose Stack

A lead magnet is NOT just a "free gift." It serves 4 simultaneous functions:

1. **Contact capture** — exchanges value for an email/phone/DM follow
2. **Problem revelation** — shows them a bigger problem that only your core offer solves
3. **Trust deposit** — proves you can actually help before money changes hands
4. **ICP filter** — only people with the right problem invest time in the right magnet

```
Narrow problem solved (free) → Next bigger problem revealed → Core offer solves it (paid)
```

**Example chain:**
- "What is my home worth?" [lead magnet: free valuation] → "I need to actually sell the house" [core offer: listing agent]
- "How many calories should I eat?" [lead magnet: macro calculator] → "I can't stay consistent alone" [core offer: coaching]
- "What's my ad spend ROI?" [lead magnet: calculator] → "My ROAS is broken — I need a system" [core offer: agency]

## 3 Types of Lead Magnets

### Type 1 — Awareness Magnet
**Purpose:** Reveals a problem the prospect didn't know they had.
- Audit / Assessment / Quiz
- Research report or benchmark
- "X Signs You Have [Problem]" diagnostic
- Industry benchmark comparison

**Best for:** Prospects who don't yet know they need what you sell.

### Type 2 — Trial / Sample
**Purpose:** Lets them experience the core offer result at a small scale.
- Free chapter / module / lesson (if you sell courses)
- Free first session / consultation
- Free 7-day trial (SaaS)
- Free "lite" version of the paid service

**Best for:** Offers where experiencing the value is the best way to sell it.

### Type 3 — One Step in a Multi-Step Process
**Purpose:** Delivers value from Step 1 of your process; the rest requires the full offer.
- Step 1 of 6-step system (delivered free)
- One deliverable from a multi-deliverable package
- Template for one part of a larger workflow
- First module of a course

**Best for:** Complex services with a clear process — proves the method works.

## 6 Lead Magnet Formats (Ranked by Consumption Effort — Low to High)

### Format 1 — Checklist
**Consumer effort:** Minimal (can be used in 5 minutes)
**Best for:** Processes, launch prep, compliance, decision frameworks
**Examples:**
- "17-Point Landing Page Audit Checklist"
- "Pre-Hire Checklist for Your First Sales Rep"
- "Monthly Offer Review Checklist"

**Delivery:** PDF, Notion template, Google Doc

### Format 2 — Template
**Consumer effort:** Minimal (fill in the blanks)
**Best for:** Anything repeatable: emails, proposals, scripts, SOPs, plans
**Examples:**
- "Cold Outreach Setter Script Template"
- "90-Day Onboarding Plan Template"
- "Weekly Sales Pipeline Review Template"

**Rule:** The template should save at least 3 hours of work. If it saves less, it's not valuable enough.

### Format 3 — Calculator / Tool
**Consumer effort:** Low (enter inputs, get outputs)
**Best for:** ROI analysis, pricing, sizing, comparisons — anything with measurable output
**Examples:**
- "LTV:CAC Calculator"
- "Price Point Optimizer"
- "Ad Spend ROI Calculator"
- "Value Equation Scorer"

**Why it wins:** Tools have a "wow moment" — the user sees their number and realizes the gap. That gap is your sales opportunity.

### Format 4 — Case Study
**Consumer effort:** Low-medium (read 5–15 minutes)
**Best for:** B2B buyers who need proof before committing, identical ICP matches
**Requirements (for maximum belief transfer):**
- Named client (not "a real estate agent in Texas")
- Specific numbers (not "significant revenue growth")
- Specific timeframe ("in 67 days")
- Identical ICP to the prospect
- Transformation arc: before → method → after

**See:** `references/proof-hierarchy.md` for full proof scoring criteria

### Format 5 — Mini-Course (3–5 Lessons)
**Consumer effort:** Medium (30–90 minutes total)
**Best for:** Education-led sales, high-ticket offers requiring belief in method, B2B authority building
**Structure:**
- Lesson 1: The problem (make it hurt)
- Lesson 2: Why current approaches fail
- Lesson 3: The new framework / first step
- Lesson 4: What's possible (proof)
- Lesson 5: CTA to core offer

**Rule:** Never teach everything. Teach the "what" and "why" — sell the "how" (the implementation).

### Format 6 — Free Community Tier
**Consumer effort:** Medium (ongoing participation)
**Best for:** Network-effect businesses, high-ticket offers, community-driven products, Skool / Discord models
**Mechanics:**
- Free tier gives basic community access
- Paid tier unlocks premium content, live calls, direct access
- Network effect increases value of paid tier over time

**Warning:** Community lead magnets require daily maintenance. Only activate if you will be present.

## Format Selection Matrix

| ICP Sophistication | Offer Price | Recommended Format |
|---|---|---|
| Low awareness / problem unaware | Any | Awareness magnet (audit/quiz) |
| Problem aware, solution unaware | < $5k | Checklist or template |
| Problem aware, solution aware | $5k–$25k | Calculator + case study |
| Solution aware, not believing in YOU | $25k+ | Mini-course or case study |
| Highly sophisticated buyer | Any | Trial / sample (let them experience) |

## MAGIC Naming Formula (Applied to Lead Magnets)

```
[M] Magnetic reason to act now
[A] Avatar (who it's for — be specific)
[G] Goal (the exact outcome it delivers)
[I] Time interval (how fast they get the result)
[C] Container word (what type of thing it is)
```

**Examples:**

| Weak Name | MAGIC Name |
|---|---|
| "Free Lead Gen Guide" | "The 15-Minute Lead Audit That Shows B2B Founders Their Fastest Path to 10 Qualified Calls This Week" |
| "Marketing Checklist" | "The 23-Point Content Machine Checklist for Service Business Owners Who Want Consistent Leads Without Paid Ads" |
| "Business Calculator" | "The 3-Input LTV:CAC Calculator That Tells SaaS Founders Exactly When to Turn On Paid Ads" |

**Rule:** If you can swap out your ICP name and it still sounds generic, it's not specific enough.

## 7-Step Design Checklist

- [ ] **Step 1:** Narrow problem defined (specific enough to solve in one interaction)
- [ ] **Step 2:** Next problem identified (the one your core offer solves)
- [ ] **Step 3:** Type selected: awareness / trial-sample / one-step
- [ ] **Step 4:** Format chosen from the 6 options above (lowest effort possible)
- [ ] **Step 5:** MAGIC name written (passes the swap test)
- [ ] **Step 6:** Consumption time < 20 minutes (or effort feels worth it)
- [ ] **Step 7:** Single CTA — one next step, no options, no ambiguity

**Quality test:** "Would my ICP pay $50 for this?" If no → add more value. If yes → you have a Grand Slam lead magnet.

## Lead Magnet Delivery Mechanisms

| Mechanism | Best Format | Friction Level |
|---|---|---|
| Instant PDF download | Checklists, templates, case studies | Very low |
| Landing page → email opt-in → auto-deliver | Any format | Low |
| DM / chat auto-reply | Short checklists, links | Very low |
| Free Skool / Discord invite | Community magnets | Low |
| Calendly → free call | High-ticket trial/consultation | Medium |
| SaaS free trial link | Tool / calculator | Low |

**Rule:** Delivery friction must be proportional to the perceived value. High friction = the magnet must be exceptional quality.

## Lead Magnet Performance Benchmarks

| Metric | Benchmark | Action If Below |
|---|---|---|
| Opt-in rate (paid traffic) | ≥ 35% | Rewrite headline and name |
| Opt-in rate (organic) | ≥ 50% | Rewrite CTA copy |
| Consumption rate | ≥ 60% | Shorten and simplify |
| "More help" CTA click-through | ≥ 15% | Audit bridge to core offer |
| Demo/call booking from magnet | ≥ 5% | Improve the bridge CTA |
