## What This Skill Does

When installed, this skill enables Claude to act as a Hormozi-trained business co-founder — analyzing offers, building lead gen systems, writing hooks, breaking down markets, and scaling operations using the same frameworks behind hundreds of millions in documented revenue.

### 6 Core Workflows

| Workflow | What It Does |
|---|---|
| **Value Equation Scorer** | Score any offer 1–1,000,000 and identify the #1 lever to improve it |
| **Grand Slam Offer Builder** | Build irresistible offers using the 6-layer stack from scratch |
| **Lead Magnet Generator** | Design lead magnets + full Core Four acquisition strategy |

| **Market Breakdown System** | TAM/SAM/SOM + competitive gaps + monopoly positioning (Thiel) |
| **Scale OS** | Audit, automate, hire, and systematize for 10x growth |

## Value Equation Quick Reference

```
Score = (Dream × Likelihood) / (Time × Effort)

Perfect = 1,000,000
= (Dream: 100 × Likelihood: 100) / (Time: 0.1 × Effort: 0.1)
```

| Score | Rating | Action |
|---|---|---|
| 1,000,000+ | 🏆 Grand Slam | Scale immediately |
| 500,000–999,999 | ⭐⭐⭐ Elite | Minor tweaks, then scale |
| 100,000–499,999 | ⭐⭐ Strong | 1-2 improvements needed |
| 20,000–99,999 | ⭐ Decent | Solid but leaving money on table |
| 5,000–19,999 | ⚠️ Weak | Significant rework before scaling |
| < 5,000 | 🚨 Broken | Rebuild before ANY marketing spend |

---

## The Fastest Levers

1. **Add a strong guarantee** → Lifts Likelihood Score dramatically
2. **Add done-for-you element** → Drops Effort Score (multiplies value)
3. **Add a 7-day quick win** → Drops Time Score (multiplies value)
4. **Niche down the ICP** → Lifts Dream Score (specificity = believability)
